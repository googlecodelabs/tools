In this tutorial we will be using some basic commands of Git. We will be working with Github as the place where our repository can be accessed

Before starting

  • Be sure that you have a Github account and you are signed in.
  • Be sure you have installed Git
  • You can use Visual Studio Code to edit the files

First steps

  • Go to https://github.com/dalthviz/git_web_practice
  • Click the fork button
  • You now should have a new repository in your account called git_web_practice

For this tutorial we have the support of a Github App to do some actions over the repository. We will need you to grant access to the repository you created (git_web_practice)

Grant Github App Access to the repository

  • Install the Github App available at https://github.com/apps/example-actions-bot
  • Click
  • Select your account (where you have your git_web_practice repository)

  • Select and select the git_web_practice repository. Click

Now that everything is setup we can start. In this case you are going to clone the repository. For this you need to open a terminal or CMD that can execute git commands. To check this you can try to execute git -v

Cloning...

  • In your git_web_practice repository check for
  • Copy the URL that is given. You can use the (this button will copy to the clipboard the URL we need).
  • In your terminal type git clone <URL from your repository>. As an example, the command should similar to this git clone https://github.com/dalthviz/git_web_practice.git
  • Do a change of directory (cd) to get to the repository root directory. Doing cd git_web_practice should be enough,

Now that we have a local copy of the repository you can make changes to it. In this case the repository just has a simple static page. The idea is to fix the website and do some changes to it while creating a commit per item listed above. Also after each commit you need to make a push of your changes.
